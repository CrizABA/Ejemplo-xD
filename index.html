Webscraping

Web Sraping es una tecnica utilizada por medio de programas para extraer informacion de distitnos sitios web, esto con fines de extraer contenido y datos mostrando su codigo HTML subyacente con el, los datos almacenados en una base de datos

Tecnicas de visualizacion

Las tecnicas de visualizacion de datos pueden ser una representacion grafica de la informacion y los datos por el uso de elementos visuales, como pueden ser graficas y mapas, esto para comprender las tendencias, valores y atiopicos

Referencia Ejemplo el Google Collab

Medias de tendencia central

Las caracteristicas globales de un conjunto de datos estadisticos pueden resumirse mediante una serie de cantidades numericas representativas llamadas parametros estadisticos

Referencia Ejemplo con Google Collab

Medidas de Dispersion

Las medidas de dispersion indican que tanto se dispersan los datos con respecto a su media aritmetica siendo que el rango es una medida de dispersion para una serie de datos no grupados igual a la diferencia del dato mayor y el dato menor
